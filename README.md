# School Performance & Education Data Analysis

## Overview

This project analyzes district-wide school performance and standardized testing data using Python, Pandas, and Jupyter Notebook.

The goal of the analysis is to identify trends in academic performance across schools by examining factors such as school type, budget allocation, student population size, and standardized test scores. The project demonstrates data cleaning, aggregation, statistical analysis, and data reporting techniques commonly used in educational and public-sector analytics.

---

# Project Objectives

The primary goals of this project are to:

- Analyze district-wide student performance data
- Compare school performance metrics across multiple schools
- Evaluate the relationship between school funding and academic outcomes
- Identify high-performing and low-performing schools
- Explore how school size and school type impact student success
- Generate clear summary tables and analytical insights using Pandas

---

# Dataset

The dataset includes district-wide information such as:

- Student math scores
- Student reading scores
- School budgets
- School size and type
- Per-student spending
- Grade-level performance data

The analysis combines both school-level and student-level datasets to generate district-wide performance summaries.

---

# Analysis & Features

## District-Level Analysis

Created high-level district summaries including:

- Total schools
- Total students
- Total district budget
- Average math and reading scores
- Passing percentages for math and reading
- Overall student passing rates

---

## School Performance Analysis

Generated detailed school-level performance reports including:

- School type
- Student population
- Budget allocation
- Per-student spending
- Average test scores
- Passing percentages

Identified:

- Highest-performing schools
- Lowest-performing schools

---

## Grade-Level Performance

Analyzed average math and reading scores by grade level for each school to compare academic trends across grades.

---

## Spending & Resource Analysis

Examined how school spending impacts academic performance by grouping schools into spending ranges and comparing:

- Average scores
- Passing rates
- Overall academic performance

---

## School Size & Type Analysis

Compared performance metrics across:

- Small, medium, and large schools
- Charter vs. district schools

This analysis helped identify patterns related to school structure and student outcomes.

# Key Insights

Some major findings from this analysis include:

- Smaller and charter schools generally demonstrated higher passing rates.
- Increased spending per student did not always correlate with stronger academic performance.
- School size appeared to have a measurable impact on overall student outcomes.
- Reading scores were generally higher and more consistent than math scores across the district.
